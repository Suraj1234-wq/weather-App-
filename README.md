🌦️ Weather App

A simple and responsive Weather Application built using HTML, CSS, and JavaScript that shows real-time weather data using an API.

🚀 Features
🌍 Search weather by city name
🌡️ Display temperature
☁️ Weather condition (Cloudy, Sunny, Rainy)
💨 Wind speed
💧 Humidity
📱 Fully responsive (Mobile + Desktop)
🛠️ Technologies Used
HTML5
CSS3 (Flexbox + Responsive Design)
JavaScript (ES6)
Weather API
